# 🎂 Birthday Surprise Website

A romantic interactive birthday website built with HTML, CSS, and JavaScript.
Created as a special surprise experience with animations, memories, music, countdown timer, and a heartfelt letter.

🔗 **Live Demo**

* Vercel: https://your-vercel-link.vercel.app
* GitHub Pages: https://your-username.github.io/repo-name/

---

## ✨ Features

* 💌 Pre-screen interactive question flow
* 🌹 Floating rose petals animation
* ✨ Sparkle background effects
* 🎵 Music player with visualizer
* 📸 Memory gallery with fullscreen preview
* ⏳ Relationship countdown timer
* ⌨️ Typewriter love letter animation
* 🎉 Confetti celebrations
* 💖 Secret final message reveal
* 📱 Fully responsive (mobile, tablet, desktop)

---

## 🛠 Tech Stack

* HTML5
* CSS3 (Glassmorphism, animations, responsive design)
* Vanilla JavaScript
* Canvas-Confetti library

## 🌍 Deployment

### ▶ Deploy on Vercel

1. Go to https://vercel.com
2. Import your GitHub repository
3. Click **Deploy**
4. Done 🎉

No build settings needed since it’s a static site.

---

### ▶ Deploy on GitHub Pages

1. Push code to GitHub
2. Go to **Repository → Settings → Pages**
3. Source: `main` branch / root folder
4. Save and wait for link generation
